Flow Timer Web App Privacy Policy

0. Purpose of Flow Timer:

The primary purpose of the Flow Timer is to provide Users with alerts regarding their work and rest intervals.

1. Integration with Toggl Track:

For Users who also utilize Toggl Track software to log their work intervals, Flow Timer offers the optional feature of saving time entries directly from the Flow Timer Web App. This functionality is achieved by leveraging the Toggl Track API service, in compliance with Toggl Track's Terms of Service and Privacy Policy.

2. Data Collection:

Flow Timer may collect and utilize a piece of sensitive User data if and only the user chooses to use this optional feature. The following sections will explain what data is collected and how it is used.

3. Collected Data:

The collected data includes an API token, consisting of a string of numbers and letters, which allows access to the Toggl Track service through other software (Flow Timer).

4. Collection Process:

Users explicitly provide this API token by filling out the respective form. Users are expected to copy a valid token from their Toggl Track Profile page beforehand.

5. Usage of Data:

The API token is EXCLUSIVELY used to authenticate Users on the Toggl Track service for the purpose of saving time entries and related information.

6. Storage Location:

The API token is stored SOLELY within the User's browser, utilizing the Storage API. This ensures that the User does not need to re-enter the token for each interaction with the Toggl Track API. The token is NOT stored by the developers of the Flow Timer or any other party.

7. Data Transmission:

The API token is NOT transferred anywhere besides the Toggl Track endpoint (servers) for the purpose of authorization (via encrypted connection). It is NOT transmitted to the developers of the Flow Timer or any other party.

8. Deletion Process:

Users can delete the API token by clicking the button labeled 'Exit from Toggl' or its localized equivalent. This action removes the token from storage within the User's browser.
